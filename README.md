# 2023_welcome_party
멋쟁이사자처럼 11기 금오공과대학교_23학번을 위한 신입생 환영회 & 동아리 박람회 안내 페이지

## 역할 및 기능
|서비스|역할 및 기능|
|---|---|
|메인|페이지 메인을 표시함|
|동아리 부스|박람회 위치와 동아리별 부스 위치를 알려줌|
|시간표|해당 기간동안 진행되는 행사 시간표를 알려줌|
|방명록|유저들이 방명록을 남길 수 있음|

## 기술스택
- React
- Django
- figma
- MySQL
